Fanta Web App

A lightweight front-end project developed using HTML, CSS, and JavaScript, showcasing smooth animations implemented with GSAP (GreenSock Animation Platform).

Overview

This project presents a simple yet visually engaging web page enhanced with scroll-based and motion animations. It is built using core web technologies, with GSAP used to create fluid and interactive animation effects that improve the overall user experience.

Key Features

Responsive Design – Works well across different screen sizes

GSAP Animations – Smooth and performant animations for UI elements

Technologies Used

HTML – Defines the structure of the web page

CSS – Handles styling and layout responsiveness

JavaScript – Adds interactivity and animation control

GSAP – Manages timeline-based animations

Getting Started

Follow these steps to run the project locally:

Clone the repository:

git clone https://github.com/abdulpara5252/fanta-main.git


Navigate to the project directory:

cd fanta-main


Open index.html in any modern web browser.

GSAP CDN

The project uses GSAP via CDN:

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.5/gsap.min.js"></script>

Usage

Open the index.html file

Scroll through the page to see GSAP-powered animations in action
